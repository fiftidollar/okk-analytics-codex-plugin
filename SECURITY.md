# Security policy

Do not open public issues containing credentials, OAuth tokens, customer data
or access-control bypass details. Report vulnerabilities privately to the Alpes
maintainers and include the affected version, reproducible impact and a minimal
proof without production data.

Supported releases are the latest tagged version and the current `main` branch.
Token/session compromise, ACL leakage, redirect-URI bypasses and any apparent
write path are treated as high priority.
